History of R

R was created by Ross Ihaka and Robert Gentleman at the University of Auckland, New Zealand, in the early 1990s. It was inspired by the S language and has become one of the most popular languages for statistical analysis and data science. Since then, R has continuously evolved with contributions from a vast community of developers worldwide.

Basic Syntax

The syntax of R is relatively simple and intuitive, making it accessible for beginners. Here are some basic examples:

  • Assignment: x <- 10
  • Math Operations: sum <- x + 5
  • Creating Vectors: vec <- c(1, 2, 3)
  • Functions: mean(vec)

Variables

In R, variables are used to store data values. You can create a variable by using the assignment operator <- or =.

x <- 5
y = 10
z <- x + y

Functions

Functions in R are used to perform specific tasks. You can create a function using the function keyword:

add <- function(a, b) {
  return(a + b)
}

result <- add(5, 3)

Data Structures

R provides various data structures to store data, such as vectors, lists, matrices, data frames, and factors. Each structure serves a specific purpose and can be manipulated using built-in functions.
