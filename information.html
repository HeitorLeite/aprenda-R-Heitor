História de R

R foi criado por Ross Ihaka e Robert Gentleman na Universidade de Auckland, Nova Zelândia, no início dos anos 1990. Foi inspirado na linguagem S e se tornou uma das linguagens mais populares para análise estatística e ciência de dados. Desde então, o R evoluiu continuamente com contribuições de uma vasta comunidade de desenvolvedores em todo o mundo.

Sintaxe Básica

A sintaxe do R é relativamente simples e intuitiva, tornando-o acessível para iniciantes. Aqui estão alguns exemplos básicos:

  • Atribuição: x <- 10
  • Operações matemáticas: soma <- x + 5
  • Criando Vetores: vec <- c(1, 2, 3)
  • Funções: média(vec)

Variáveis

Em R, variáveis ​​são usadas para armazenar valores de dados. Você pode criar uma variável usando o operador de atribuição <- ou =.

x <- 5
y = 10
z <- x + y

Funções

Funções em R são usadas para executar tarefas específicas. Você pode criar uma função usando a function palavra-chave:

adicionar <- função(a, b) {
retornar (a + b)
}

resultado <- add(5, 3)

Estruturas de dados

R fornece várias estruturas de dados para armazenar dados, como vetores, listas, matrizes, quadros de dados e fatores. Cada estrutura serve a um propósito específico e pode ser manipulada usando funções integradas.
