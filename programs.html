Introduction to R Programs

R is used in various domains, including data analysis, statistical modeling, and machine learning. Here are some examples of programs you can create with R:

Example Programs

Data Analysis

This example demonstrates how to load a dataset and perform basic data analysis using R.

# Load the dataset
data <- read.csv("data.csv")

# Summary of the dataset
summary(data)

# Plot a histogram of a column
hist(data$column_name)

Statistical Modeling

This example shows how to fit a linear model to a dataset.

# Load the dataset
data <- read.csv("data.csv")

# Fit a linear model
model <- lm(y ~ x, data=data)

# Summary of the model
summary(model)

Machine Learning

This example demonstrates how to build a simple machine learning model using the caret package.

# Load the caret package
library(caret)

# Load the dataset
data <- read.csv("data.csv")

# Train a model
model <- train(y ~ x, data=data, method="rpart")

# Print the model
print(model)
