Guia de Instalação Passo a Passo

  1. Baixe o R

    Visite o site oficial do R em CRAN.

  2. Selecione Seu Sistema Operacional

    Escolha o link de download apropriado para o seu sistema operacional (Windows, macOS ou Linux).

  3. Instale no Windows

    • Baixe o instalador do R para Windows.
    • Execute o instalador e siga as instruções na tela.
    • Escolha as configurações padrão para a maioria dos usuários.

  4. Instale no macOS

    • Baixe o arquivo .pkg para macOS.
    • Abra o arquivo e siga as instruções na tela para instalar.

  5. Instale no Linux

    • Abra o terminal.
    • Adicione o repositório CRAN ao seu sistema de gerenciamento de pacotes.
    • Atualize a lista de pacotes do sistema.
    • Instale o R usando o gerenciador de pacotes do seu sistema.
    # Comandos para Debian/Ubuntu
sudo apt update
sudo apt install r-base
# Comandos para Fedora
sudo dnf install R
# Comandos para Arch Linux
sudo pacman -S r

  6. Verifique a Instalação

    Abra o VSCode vá em extensões, digite R na barra de pesquisa e baixe a primeira extensão que apareceu.

    Agora volte a sua pasta de arquivos, crie um arquivo chamado test.r e escreva o comando print("Hello World"), clique no botao de executar e caso o comenda apareça no Prompt significa que funcionou.
